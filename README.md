# :zap: Flask-Weather-API

[https://pravzzyy-flask-weather-api-1.glitch.me/](https://pravzzyy-flask-weather-api-1.glitch.me/)

## :wrench: Features
- Temperature
- Humidity
- Wind Speed

- Flask
